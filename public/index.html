    Roteiro de Instalação: Zabbix Agent 2
    
                    
    Como habilitar o monitoramento de saúde nos seus servidores Sankhya.
    
                
    
                
            
    
            
            
    O monitoramento de hardware (CPU, RAM, Disco) e serviços de banco do ERP exige a presença do **Zabbix Agent 2** ativo na máquina local.
    
            
            
     Windows (Servidores AD ou Web)
    
            
    Abra o **PowerShell** como Administrador no servidor Sankhya Windows e execute o comando rápida de implantação em lote:
    
            
    Invoke-WebRequest -Uri "https://cdn.zabbix.com/zabbix/binaries/stable/7.0/7.0.0/zabbix_agent2-7.0.0-windows-amd64-openssl.msi" -OutFile "$env:TEMP\zabbix_agent2.msi"
Start-Process msiexec.exe -ArgumentList "/i `\"$env:TEMP\zabbix_agent2.msi`\" /qn SERVER=`\"192.168.100.96`\" SERVERACTIVE=`\"192.168.100.96`\" HOSTNAME=`\"$env:COMPUTERNAME`\" ENABLEPATH=1" -Wait
Start-Service "Zabbix Agent 2"
    

            
     Linux (Servidores de Produção)
    
            
    Acesse via SSH e execute os comandos rápidos de repositório e instalação (Ubuntu Server):
    
            
    sudo apt update && sudo apt install zabbix-agent2 -y
sudo sed -i 's/Server=127.0.0.1/Server=192.168.100.96/g' /etc/zabbix/zabbix_agent2.conf
sudo sed -i 's/ServerActive=127.0.0.1/ServerActive=192.168.100.96/g' /etc/zabbix/zabbix_agent2.conf
sudo systemctl restart zabbix-agent2 && sudo systemctl enable zabbix-agent2
    

            
    
                
                
    
                    Fase do Zabbix Server: Após rodar os comandos no servidor local, certifique-se de associar o template Windows by Zabbix agent ou Linux by Zabbix agent no Host correspondente dentro do painel web do seu Zabbix Server!
